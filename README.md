# 🎬 AuraCinema AI

An AI Mood Movie Recommender website built using Flask and TMDB API.

## ✨ Features

- 😊 Mood-based movie recommendations
- 🔍 Search movies by name
- 🎞️ Real movie posters and descriptions
- ⭐ Movie ratings and release dates
- 🌌 Aurora-themed user interface

## 🛠️ Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript
- TMDB API

## 🚀 How to Run the Project

1. Install required libraries:
pip install flask requests

2. Run the application:
python app.py

3. Open in browser:
http://127.0.0.1:5000⁠�

## 📌 Project Purpose

AuraCinema AI recommends movies based on the user's mood and allows users to search for movies using real-time movie data from TMDB API.
